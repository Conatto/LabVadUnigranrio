O que um Display de Caracteres faz?

Um display de cristal líquido, acrônimo de LCD (em inglês liquid crystal display). Mostra dados lidos pelo LabVad em letras e números, de uma forma inteligível.

Como funciona no LabVad Unigranrio?

O display fica conectado em pinos e o código deve ser feito de acordo com essa conexão.

Exemplo de código: 

 
/* Display de Caracteres
    * ------------
// Incluir a biblioteca LCD:
    #include 
// Inicializar a biblioteca com os números dos pinos digitais
    LiquidCrystal lcd(12, 11, 5, 4, 3, 2);
    void setup() {
    // Configurar o número de colunas e linhas do LCD:
    lcd.begin(16, 2);
    // Texto a ser mostrado no LCD (texto fixo):
    lcd.print("LABVAD UNIGRANRIO");
}
void loop() {
    // Posicionar o cursor na coluna 0 e linha 1 (segunda linha):
    lcd.setCursor(0, 1);
    // Imprimir contagem de segundos decorridos:
    lcd.print(millis() / 1000);
}
